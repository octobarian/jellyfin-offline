About

RV Media Player is a self-hosted media player built for Raspberry Pi. It connects to your Jellyfin server to sync your movie and TV library, lets you download titles to local storage, and plays them back through VLC - with or without an internet connection.

The web interface runs on the Pi and is accessible from any browser on your local network. Once content is downloaded, the entire experience is fully offline.

What it does

  • Browse your full Jellyfin library from any device on the network
  • Download movies and shows to a local drive or USB mount for offline use
  • Play local files in VLC with one click - normal window with controls
  • Stream directly from Jellyfin through VLC when online
  • Tracks download progress in real time with poster art
  • Works in offline mode - local media is always accessible
  • Configurable media storage path, including USB mounts like /mnt/stor
  • Runs as a systemd service and launches automatically on boot

Setup

  1. Clone the repo and run sudo ./install.sh
  2. The installer prompts for your Jellyfin URL, username, API key, and media storage path
  3. A desktop shortcut is created at ~/Desktop/RV Media Player
  4. The app is available at http://localhost:5000 or http://<pi-ip>:5000

Requirements

  • Raspberry Pi running Raspberry Pi OS (Bullseye or Bookworm)
  • A running Jellyfin server with an API key
  • VLC Media Player (installed automatically by install.sh)
  • Local storage or a USB drive for downloaded media

Troubleshooting

  • Check the system status page for service health
  • Verify your Jellyfin credentials in Settings
  • Run the VLC path test in Settings if playback does not launch
  • Application logs are written to the logs/ directory
